### Setup

```
npm install --global yarn
yarn --version
node -v
npm -v


npx create-react-app starter-app --template typescript

cd starter-app
yarn start


cd frontend 
yarn start

yarn add @mui/material @emotion/react @emotion/styled
yarn add @mui/icons-material
yarn add react-router-dom
```
